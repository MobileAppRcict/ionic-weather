# ionic2-weather

Simple Ionic 3 application to show weather forecast with dark sky api.

## Instalation

Install node modules:

```
npm install -g ionic cordova
```

Create .env file with:

```
DARK_SKY_API_KEY=YOUR_DARK_SKY_API_KEY
```

Start preview in browser:

```
ionic serve
```
